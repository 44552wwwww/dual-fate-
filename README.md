## 📖 算命方法

本项目使用中国传统五术中的两套核心命理体系：

### 🔮 子平术（八字命理）

**子平术**，又称八字、四柱预测学，是宋代命理学家徐子平创立的核心命理学流派。

**推算原理：**
- 将出生年、月、日、时转换为四组**天干地支**（共八个字），即"四柱八字"
- 以**日柱天干**代表命主本人，分析其他七个字与日主的**五行生克**关系
- 通过十种关系（**十神**）——正印、偏印、正官、七杀、正财、偏财、食神、伤官、比肩、劫财——推断性格、事业、婚姻、财运
- 根据**身强身弱**确定**用神**（有利五行）和**忌神**（不利五行）
- 排**大运**（每十年一换的运势周期），结合流年看运势起伏

**核心法则：** 五行平衡——"身强用克泄耗，身弱用生扶"。配合**调候**（季节冷暖）、**通关**（五行相战时调和）等方法确定用神。

---

### 🌟 紫微斗数

**紫微斗数**，相传为宋代陈抟（陈希夷）所创，是八字之外另一大主流终身命理体系。

**推算原理：**
- 根据出生月、时确定**命宫**和**身宫**（两个核心定位宫位）
- 以命宫为中心排出**十二宫**：命宫、兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、交友宫、官禄宫、田宅宫、福德宫、父母宫
- 根据农历生日和五行局安**紫微星**，进而安十四主星、辅星、煞星共数十颗星曜
- 每颗星曜在不同宫位有**庙旺利陷**不同的力量状态
- 通过**四化飞星**（化禄、化权、化科、化忌）判断运势的引动和变化
- 排**大限**（十年一换的宫位推移），看各阶段人生重心

**核心法则：** 星宫互参——同一颗星在不同宫位含义截然不同。命好不如限好，本命星盘看底色，大限流年看时机。

---

### ⚖️ 双鉴：两套体系交叉验证

八字和紫微斗数使用**完全不同的推算方法**，是两套独立的命理体系：

| 对比 | 八字（子平术） | 紫微斗数 |
|------|--------------|----------|
| 理论基础 | 干支五行学 | 星象学、天文学 |
| 核心元素 | 八个字（天干地支） | 一百多颗星曜 |
| 推演方式 | 五行生克、十神分析 | 安星布宫、星宫互参 |
| 空间维度 | 四柱（年月日时） | 十二宫 |
| 信息载体 | 抽象的天干地支 | 形象的星曜 |

当两套独立体系得出**相同结论**时，可信度更高。当结论不同时，可能揭示了问题的不同侧面。

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/44552wwwww/命运双鉴.git
cd 命运双鉴/scripts

# 生成报告
python generate_report.py 2006 4 10 1 男
# → 命运双鉴_20060410_男.html
```

生成的 HTML 文件完全自包含（内嵌 CSS/JS），直接发给任何人都能用浏览器打开。

## ✨ 特点

- **零依赖** — 纯 Python 标准库，无需 pip install
- **全动态生成** — 所有分析文字根据实际命盘数据实时计算，无硬编码
- **跨平台** — Windows / macOS / Linux 均可运行
